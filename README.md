# StatusBar

The status bar at the top displays the current status of the system, such as time, system tray, etc.

## Dependencies

```bash
sudo apt install libkf5windowsystem-dev -y
```

## Build
or run build.sh


```bash
cmake -DCMAKE_INSTALL_PREFIX:PATH=/usr ..
make
sudo make install
```
